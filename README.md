# vic80_zx_keyboard: Контроллер клавиатуры PS/2 и джойстиков Sega MD для ZX Spectrum

Аппаратный контроллер для подключения современной клавиатуры стандарта PS/2 и двух классических джойстиков от Sega Mega Drive (Sega MD) к компьютерам ZX Spectrum.

Проект построен на базе популярного и доступного микроконтроллера **ATmega8** и аналогового коммутатора **MT8816** (матрицы ключей).

## ⚡ Главная фишка: Безвейтовая архитектура (No WAIT)
Большинство старых или простых контроллеров клавиатуры на микроконтроллерах вынуждены тормозить процессор Спектрума сигналом `WAIT`, чтобы AVR-ка успела среагировать на чтение порта `#FE`.

В этой разработке опрос PS/2 клавиатуры и джойстиков вынесен в фоновый режим. Микроконтроллер ATmega8 транслирует состояние клавиш на аналоговую матрицу **MT8816**, которая физически замыкает нужные линии строк и столбцов. Для шины Z80 контроллер выглядит как оригинальная механическая клавиатура. Схема работает абсолютно **без задержек процессора (Zero-WAIT)**, обеспечивая идеальные тайминги в играх и демосценах.

## 🛠 Технические характеристики
* **Ядро:** ATmega8 (работает на тактовой частоте 16 МГц от внешнего кварца).
* **Коммутатор:** MT8816 (микросхема аналогового кросс-коммутатора 8х16).
* **Ввод:** Клавиатура PS/2.
* **Игровые порты:** 2 порта для джойстиков Sega Mega Drive (считываются все основные кнопки).
* **Раскладка:** Полная эмуляция стандартной матрицы ZX Spectrum + поддержка расширенных клавиш.

## 💾 Структура репозитория
* `/Schematics` — Принципиальная схема устройства в форматах PDF и PNG.
* `/PCB` — Файлы печатной платы: Gerber-архив в ZIP для производства и изображения слоев.
* `/Firmware` — Файлы для прошивки микроконтроллера:
  * `vic80_zx_keyboard.hex` — Память программ (Flash).
  * `vic80_zx_keyboard.eep` — Файл стартовой конфигурации пресетов (EEPROM).

## ⚙️ Настройка Fuse-битов при прошивке

Для стабильной работы и синхронизации микроконтроллер должен тактоваться от внешнего кварцевого резонатора в режиме максимальной амплитуды (Large Output Swing), а также иметь включенный аппаратный сброс по питанию (BOD 4.0V) для предотвращения порчи пресетов в EEPROM.

**Каноничные значения фьюзов для прямой прошивки через консоль `avrdude`:**
* **Low Fuse:** `0xC1`
* **High Fuse:** `0xFC` (или `0xD0`, если отключается Bootloader)

⚠️ **ВНИМАНИЕ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ AVR8 Burn-O-Mat:**
В графическом Hex-редакторе этой программы текстовые подписи полей перепутаны местами! 
* Там, где написано **`hfuse`**, программа настраивает нижний байт (задавайте туда `C1`).
* Там, где написано **`lfuse`**, настраивается верхний байт (задавайте туда `3F` с учетом софтовой маски).
На вкладке *Oscillator Options* выбирайте **strictly `slowly rising power`**. Пункт *«brown out detection enabled»* выбирать **НЕЛЬЗЯ**, иначе чип перейдет в режим внешнего генератора и заблокируется!

⚠️ **КРИТИЧЕСКИ ВАЖНО:**
Помимо файла `.hex`, обязательно запишите файл **`vic80_zx_keyboard.eep`** в память **EEPROM** микроконтроллера. Если этого не сделать, при первом старте конфигурация джойстиков будет заполнена мусором.

*(Внимание: не забудьте отключить режим совместимости с ATmega103 (биг-фуз `M103C`), если ваш программатор выставляет его по умолчанию!)*

# vic80_zx_keyboard: Контроллер клавиатуры PS/2 и джойстиков Sega MD для ZX Spectrum

Аппаратный контроллер для подключения современной клавиатуры стандарта PS/2 и двух классических джойстиков от Sega Mega Drive (Sega MD) к компьютерам ZX Spectrum.

Проект построен на базе популярного и доступного микроконтроллера **ATmega8** и аналогового коммутатора **MT8816** (матрицы ключей).

## ⚡ Главная фишка: Безвейтовая архитектура (No WAIT)
Большинство старых или простых контроллеров клавиатуры на микроконтроллерах вынуждены тормозить процессор Спектрума сигналом `WAIT`, чтобы AVR-ка успела среагировать на чтение порта `#FE`.

В этой разработке опрос PS/2 клавиатуры и джойстиков вынесен в фоновый режим. Микроконтроллер ATmega8 транслирует состояние клавиш на аналоговую матрицу **MT8816**, которая физически замыкает нужные линии строк и столбцов. Для шины Z80 контроллер выглядит как оригинальная механическая клавиатура. Схема работает абсолютно **без задержек процессора (Zero-WAIT)**, обеспечивая идеальные тайминги в играх и демосценах.

## 🛠 Технические характеристики
* **Ядро:** ATmega8 (работает на тактовой частоте 16 МГц от внешнего кварца).
* **Коммутатор:** MT8816 (микросхема аналогового кросс-коммутатора 8х16).
* **Ввод:** Клавиатура PS/2.
* **Игровые порты:** 2 порта для джойстиков Sega Mega Drive (полноценно считываются все 6 основных кнопок + Start и Mode, всего 12 линий управления).
* **Раскладка:** Полная эмуляция стандартной матрицы ZX Spectrum + поддержка расширенных клавиш.

## 💾 Структура репозитория
* `/Schematics` — Принципиальная схема устройства в форматах PDF и PNG.
* `/PCB` — Файлы печатной платы: Gerber-архив в формате `.zip` для заказа плат на заводе и изображения слоев.
* `/Firmware` — Файлы для прошивки микроконтроллера:
  * `ZX_Key_Joy_v1.3.7.hex` — Память программ (Flash).
  * `ZX_Key_Joy_v1.3.7.eep` — Файл стартовой конфигурации пресетов (EEPROM).

## ⚙️ Настройка Fuse-битов при прошивке (Критически важно!)

Для стабильной работы микроконтроллер должен тактоваться от внешнего кварцевого резонатора на 16 МГц в режиме максимальной амплитуды (Large Output Swing), а также иметь включенный аппаратный сброс по питанию (BOD 4.0V) для защиты сохраненных профилей в EEPROM от порчи при выключении Спектрума.

Поскольку разный софт отображает фьюзы по-разному, строго следуйте этой таблице-шпаргалке:


| Метод прошивки / Программа | Значение Low Fuse | Значение High Fuse |
| :--- | :---: | :---: |
| **Каноничные биты (AVRDUDE / консоль напрямую)** | **`0xC1`** | **`0xFC`** (или `0xD0`) |
| **eXtreme Burner - AVR** *(инвертированное чтение)* | **`0x3F`** | **`0xC1`** |
| **AVR8 Burn-O-Mat** *(перепутаны поля hfuse/lfuse)* | В поле hfuse: **`C1`** | В поле lfuse: **`3F`** |

### ⚠️ Важные предупреждения по софту:
1. **Для пользователей AVR8 Burn-O-Mat:** В Hex-редакторе этой программы текстовые подписи полей перепутаны местами! Кроме того, на вкладке *Oscillator Options* выбирайте строго пункт **`slowly rising power`**. Пункт *«brown out detection enabled»* выбирать **НЕЛЬЗЯ**, иначе чип уйдет в режим внешнего генератора и «окирпичится»!
2. **Для пользователей eXtreme Burner - AVR:** Программа зеркально инвертирует биты, поэтому вбивайте фьюзы строго по таблице выше (`Low: 3F`, `High: C1`).

### 💾 КРИТИЧЕСКИ ВАЖНО ПРИ ПРОШИВКЕ:
Поскольку память программ микроконтроллера оптимизирована до предела (занято 96%), в коде нет функции автоинициализации памяти «из коробки». Поэтому помимо основного файла `.hex`, вы **ОБЯЗАТЕЛЬНО** должны записать файл **`vic80_zx_keyboard.eep`** in энергонезависимую память (**EEPROM**) микроконтроллера. Если этого не сделать, при первом старте конфигурация джойстиков будет заполнена мусором и они не будут работать.

*(Внимание: не забудьте отключить режим совместимости с ATmega103 (биг-фуз `M103C`), если ваш программатор выставляет его по умолчанию!)*
